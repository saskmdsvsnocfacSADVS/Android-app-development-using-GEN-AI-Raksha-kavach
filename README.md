## 🚀 How to Use / Install

### Option A — Open Directly on Phone
1. Transfer the ZIP to your phone (WhatsApp, email, USB, Google Drive)
2. Extract the ZIP
3. Open `index.html` in Chrome (Android) or Safari (iPhone)
4. Tap **"Add to Home Screen"** from the browser menu
5. Done! Use it like a native app ✅

### Option B — Host Online (Share with Workers)
Upload the folder to any free hosting:
- **GitHub Pages** (free): Push to a repo → enable Pages
- **Netlify** (free): Drag & drop the folder at netlify.com
- **Vercel** (free): `vercel deploy` in the folder

Once hosted, share the URL — anyone can install from their phone!

## ✨ Features

- **10 Construction Tasks** — Welding, Height Work, Electrical, Trench Dig & more
- **Task-Specific PPE Checklist** — gear changes based on selected task
- **Safety Avatar** — visual worker that upgrades as gear is checked
- **Risk Meter** — real-time risk % with injury education
- **Incident Log** — report near misses, stored offline on device
- **Daily Safety Quiz** — 8-question pool, 5 shown daily, randomized
- **Safety Score & Streak** — gamified safety behavior
- **Start-of-Day Reminder** — pops on first open each session
- **Works Fully Offline** — no internet needed after first load
- **High-Visibility UI** — Industrial Yellow/Black design

## 🛡️ Tech Stack
