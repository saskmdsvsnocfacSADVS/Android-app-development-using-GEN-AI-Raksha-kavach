📲 How to Install

🤖
ANDROID (Chrome)
Samsung, OnePlus, Xiaomi, Redmi etc.
  1. 1Open Chrome browser on your Android phone
  2. 2Open the raksha-kavach.html file (or the hosted link)
  3. 3Tap the ⋮ three-dot menu (top right corner)
  4. 4Tap "Add to Home screen" or "Install app"
  5. 5Tap Install — app icon appears on your home screen! ✅
🍎
iPHONE / iPAD (Safari)
iOS 14+ required
  1. 1Open Safari browser (must be Safari, not Chrome)
  2. 2Open the app link or HTML file
  3. 3Tap the Share button (□↑ at the bottom)
  4. 4Scroll down and tap "Add to Home Screen"
  5. 5Tap Add — Raksha-Kavach is now on your home screen! ✅
💻
WINDOWS / MAC (Chrome)
Desktop installation
  1. 1Open in Google Chrome on your computer
  2. 2Look for the install icon (⊕) in the address bar
  3. 3Click it and select "Install"
  4. 4App opens like a native app — no browser UI! ✅
